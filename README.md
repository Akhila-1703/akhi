# Todo Application

A simple and elegant Todo list app designed to help you stay organized. It features local storage so you never lose your tasks!

## Features
- Add new tasks quickly.
- Mark tasks as completed (strikethrough effect).
- Delete tasks.
- **Persistent Storage:** Tasks are saved in your browser's local storage automatically.

## How to Run
Open the `index.html` file in your browser to start tracking your tasks.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
